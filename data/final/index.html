"booru" is a web project which utilizes the Danbooru api to generate randomized 3D objects based on the images hosted on danbooru.donmai.us. Unlike imageboards based on Futaba Channel, such as 4chan, Danbooru aims for a non-hierarchical semantic structure in which users are able to post content and add tags, annotations, translations, and comments.

My intention is to transform this vast amount of bizarre information (over 2946152 anime images have been uploaded onto Danbooru as of 12/6/2017) into abstract bodies one can visually grasp as formal entities.
